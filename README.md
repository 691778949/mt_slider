# mt_slider
### 使用说明
1、git clone https://github.com/qinLess/mt_slider.git  
2、cd mt_slider  
3、npm install  
4、npm run dev  
会在本地监听7000端口  
127.0.0.1:7000/token   后台获取token接口  
127.0.0.1:7000/tbToken 滑块加密参数获取  
[示例代码](https://github.com/qinLess/mt_slider/blob/master/test/test.py)
本代码仅限于账号密码方式登陆
